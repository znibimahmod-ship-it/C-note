C++ MAIN CONCEPTS
=================

1. HELLO WORLD
   #include <iostream>
   using namespace std;
   int main() {
       cout << "Hello, World!" << endl;
       return 0;
   }

2. ESCAPE SEQUENCES
   \n = newline
   \t = tab
   \r = carriage return
   \\ = backslash
   \" = double quote

3. VARIABLE TYPES
   int - integers (-2147483648 to 2147483647)
   char - single character
   string - text/string
   bool - true or false
   float - decimal numbers (7 digits precision)
   double - decimal numbers (15 digits precision)

4. CONST KEYWORD
   const int maxAge = 120;
   Creates a constant variable that cannot be changed

5. BASIC OUTPUT
   cout << "text" << endl;  // prints with newline
   cout << value;           // prints variable value
   cout << a << " " << b;   // prints multiple values
