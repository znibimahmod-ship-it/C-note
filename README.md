## 🚀 Getting Started

### Prerequisites
- C++ compiler (g++, clang, MSVC, or MinGW)
- Text editor or IDE (VS Code, Code::Blocks, Visual Studio, etc.)

### Compile & Run

**Linux/Mac/MinGW:**
```bash
g++ -o program_name basics/hello-world.cpp
./program_name
```

**Windows (MSVC):**
```bash
cl basics/hello-world.cpp
program_name.exe
```

## 💡 Tips for Learning C++

1. **Code Along** - Don't just read, write and run the code
2. **Experiment** - Modify code and see what happens
3. **Debug** - Use print statements to understand flow
4. **Practice** - Solve problems on platforms like HackerRank, LeetCode
5. **Build Projects** - Apply what you learn in real projects
