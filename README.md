## 🚀 Getting Started

### Prerequisites
- **C++ Compiler**: g++, clang, or MSVC
- **IDE/Editor**: VS Code, Code::Blocks, Visual Studio, or any text editor
- **Git**: For version control

### Setup

1. Clone this repository:
```bash
git clone https://github.com/znibimahmod-ship-it/C-note.git
cd C-note
```

2. Compile a C++ program:
```bash
# Linux/Mac/MinGW
g++ -o output_name filename.cpp
./output_name

# Windows (MSVC)
cl filename.cpp
filename.exe
```

3. For competitive programming compilation with optimizations:
```bash
g++ -O2 -std=c++17 -Wall -Wextra filename.cpp -o output_name
```

## 💡 Key Topics Covered

### Complexity Analysis
- **Time Complexity**: O(1), O(log n), O(n), O(n log n), O(n²), O(2^n)
- **Space Complexity**: Understanding and optimizing memory usage
- **Trade-offs**: Time vs Space considerations

### Problem-Solving Strategies
- **Brute Force**: Simple but inefficient approach
- **Divide & Conquer**: Breaking problem into subproblems
- **Greedy**: Making locally optimal choices
- **Dynamic Programming**: Overlapping subproblems with memoization
- **Backtracking**: Exploring all possibilities systematically
- **Two Pointers**: Efficient traversal of arrays
- **Sliding Window**: Solving subarray/substring problems

### Competitive Programming Tips
- Read input/output efficiently
- Handle edge cases carefully
- Optimize for time limits
- Use appropriate data structures
- Debug systematically
